Title: Sales Tax Calculator

Description:

The "Sales Tax Calculator" is a web application designed to assist users in calculating sales tax and the total cost of a purchase. This simple and user-friendly tool enables users to input the subtotal of their purchase and the applicable tax rate, then computes the corresponding sales tax and total cost, providing an instant breakdown of the financial aspects of a transaction.

Technologies Used:

HTML:

The structure of the web page is defined using HTML, including labels and input fields for user input, as well as headings and buttons for user interaction.

CSS:

The project utilizes an external CSS file (sales.css) to style the page, making it visually appealing and enhancing the user experience.

JavaScript:

The logic for calculating sales tax and displaying results is implemented using JavaScript. An external JavaScript file (sales_tax.js) is included to separate the code from the HTML for maintainability and organization.

Title: Test Score Array

Description:

The "Test Score Array" web application is designed to help users manage and analyze test scores. Users can input both the name of a student and their test score. This simple tool allows users to add this data to an array, display the results, and view the entered scores. It serves as a basic system for recording and organizing test scores.

Technologies Used:

HTML:

The HTML structure defines the input fields for the student's name and test score. It also includes buttons for adding data to the array and displaying the results. The structure of the page is enhanced with the use of labels and buttons.

CSS:

The project uses an external CSS file (text_score.css) for styling, making the web page visually appealing and improving the user experience by applying styles to various elements.

JavaScript:

The logic for managing the test score array, adding data, and displaying results is implemented using JavaScript. An external JavaScript file (test_scores.js) is included to maintain clean and organized code, separate from the HTML structure. JavaScript is responsible for handling user interactions and data manipulation.

Title: Complete our survey

Description:

The "Complete our survey " web page is designed to gather user input through a form. It provides a structured and user-friendly interface for users to complete a survey. The form includes various fields, such as contact information, geographic information, and a section to specify how the user heard about the survey. This page aims to streamline the data collection process and ensures that the required fields are filled out correctly, making it an effective tool for gathering user responses.

Technologies Used:

HTML:

The HTML structure defines the form and its various input fields. It includes attributes such as "autofocus" and "required" to improve the user experience by focusing on the email input and ensuring that essential fields are completed.

CSS:

The project utilizes embedded CSS to style the form, creating an appealing and well-organized layout. It specifies font styles, field dimensions, and form validation styles, making the user interface visually pleasing and user-friendly.

Form Validation:

Form validation is implemented through HTML attributes like "required," "pattern," and input types to ensure that the user provides valid data. Additionally, checkboxes are used to capture how users heard about the survey.

Title: The Alien Monster is Here

Description:

"The Alien Monster is Here" is a web page that immerses visitors in the world of alien invaders. It features an iconic alien monster image and provides information about various invaders that appear in different seasons. Technologies Used:

HTML:

The HTML structure defines the layout of the page, including the header, main content, and navigation elements. It incorporates text and image elements to present the alien invaders theme.

CSS:

The project uses external CSS files (normalize.css and style.css) for styling, ensuring a consistent and visually appealing design. CSS is responsible for controlling the page's layout and presentation.
